# Dossier des documents Markdown

Placez ici vos fichiers `.md`. Ils apparaissent automatiquement sur la page **Documents** du site.

- **URL** : chaque fichier `mon-titre.md` est accessible en `/docs/mon-titre/` (slug dérivé du nom).
- **Titre affiché** : la première ligne `# Mon titre` dans le fichier, sinon le nom du fichier est utilisé.
- **Résumé** : extrait du premier bloc de texte sous les titres.

En production, vous pouvez définir un autre répertoire avec la variable d’environnement `MARKDOWN_DOCS_DIR`.
