## Comment faire ? {#bv-1}

1. Ajoutez un fichier `nom-de-votre-page.md` dans ce dossier.
2. Utilisez une première ligne `# Titre` pour le titre affiché sur le site.
3. Rechargez la liste des documents : votre page apparaît avec les autres.

**Gras**, *italique*, listes et blocs de code fonctionnent comme d’habitude.
